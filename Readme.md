# 动态守护姬DynamicGuardian



## 是什么？

动态守护姬DynamicGuardian是一款扫码登录B站账号给指定up主的指定数量未点赞动态点赞的程序

## 如何使用？

下载[Releases](https://github.com/forSeasons333/bili_autolike/releases)下程序本地运行即可
